# Giphy search app
## Built using Gatsby.js
### with a little help from

 - Axios
 - Netlify
 - react infinite-scroll

 ### Get up and running
 ``` bash 
 npm start
 ```
 #### Will run Client on `localhost:8000`
 ``` bash
 netlify-lambda serve src/lambda
 ```
 #### will run lamda server on `localhost:9000`
